A responsive, multilingual (🇫🇷 / 🇪🇸 / 🇬🇧 / 🇦🇩) single-page application showcasing my studies, projects, skills, and résumé as an aspiring **cyber-security Pentester**. Built with **React 17** and deployed automatically to **GitHub Pages** on every push to `main`.


---

## 🗂 Project Structure

```text
├── public/              # static assets, favicon, index.html
├── src/
│   ├── assets/          # images, Lottie JSONs
│   ├── components/
│   │   ├── Home/        # hero, about, projects, tech stack …
│   │   ├── Resume/      # PDF résumé generator & timeline
│   │   ├── Navbar.js    # sticky header + language selector
│   │   └── …
│   ├── styles/          # global SCSS & CSS modules
│   ├── i18n.js          # translation configuration
│   └── App.js / index.js
└── package.json
```


## 🚀 Instalation

```bash
git clone https://github.com/Jordanmacia/jordanmacia.github.io
cd jordanmacia.github.io
npm install
npm start       # http://localhost:3000
```
